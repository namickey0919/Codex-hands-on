# Codex-hands-on

This repository contains a sample iOS application **AutoPlaylist** created with Swift and SwiftUI.

AutoPlaylist automatically builds a playlist of the top songs for your favorite artists using the iTunes Search API. You can search for artists, add them to your playlist and start shuffle playback of the preview tracks.

## Features
- Artist search using the iTunes Search API
- Add an artist to automatically fetch their top 10 tracks
- Order tracks by popularity or most recent release
- Shuffle only playback of preview audio

## Building
1. Open the `AutoPlaylist` folder in Xcode (`File > Open`).
2. Build and run the app on an iOS simulator or device running iOS 17 or later.

This is a basic example and does not include full error handling or persistent storage. Feel free to extend it for your needs.
